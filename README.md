# Makestar Homebrew Tap

Homebrew tap for Makestar public CLI packages.

Install Makestar Admin CLI on Apple Silicon macOS:

```bash
brew install makestarlab/tap/makestar-admin-cli
makestar-admin --version
```

Formula updates are generated from the public `makestar-admin-cli-releases` manifest and reviewed manually before broad rollout.

This tap installs only the `makestar-admin` CLI. Claude/Codex skills and other agent bundles are distributed separately.
